ใบกิจกรรมออนไลน์

เรื่อง นิพจน์ตรรกศาสตร์และขั้นตอนวิธี

กรอกข้อมูลส่วนตัวให้ครบถ้วนก่อนเริ่มทำกิจกรรม

ตอนที่ 1: เติมคำในช่องว่าง (ทฤษฎีและสัญลักษณ์)

คำชี้แจง: นำตัวเลือกที่กำหนดให้เติมลงในช่องว่างให้ถูกต้อง (ตัวเลือก: &&, ||, !, True, False, ==, !=)

  1. เครื่องหมายเปรียบเทียบที่ใช้ตรวจสอบว่า "ค่าสองค่าไม่เท่ากัน" คือ
  2. ตัวเชื่อมตรรกศาสตร์ที่ผลลัพธ์จะเป็น True แค่กรณีเดียวคือ ต้องเป็นจริงทั้งหมด คือ
  3. ตัวเชื่อมตรรกศาสตร์ที่ทำหน้าที่ "กลับค่าความจริง" คือ
  4. นิพจน์ (10 > 5) จะให้ผลลัพธ์เป็นค่าความจริงคือ
  5. นิพจน์ (3 != 3) จะให้ผลลัพธ์เป็นค่าความจริงคือ

ตอนที่ 2: หาผลลัพธ์สุดท้ายของนิพจน์

คำชี้แจง: จงคำนวณและเติมผลลัพธ์สุดท้ายว่าเป็น True หรือ False

ข้อที่ นิพจน์ตรรกศาสตร์ กรอกคำตอบ (True / False)
1 (5 > 2) && (3 < 7)
2 (10 == 10) || (4 > 9)
3 !(8 >= 8)
4 (5 != 5) && (2 == 2)
5 (12 < 20) || (7 != 7)

ตอนที่ 3: ขั้นตอนวิธีแบบบรรยาย (แบบทางเลือก และ แบบทำซ้ำ)

คำชี้แจง: วิเคราะห์ขั้นตอนวิธีแบบบรรยายต่อไปนี้ แล้วตอบว่าพิมพ์ผลลัพธ์สุดท้ายเป็นตัวเลขอะไร หรือเงื่อนไขเป็น True หรือ False

  1. [แบบทางเลือก] กำหนดให้คะแนนสอบ score = 75
    ขั้นตอนวิธี: "ถ้า score มากกว่าหรือเท่ากับ 80 ให้พิมพ์คำว่า True ถ้าไม่ใช่ให้พิมพ์คำว่า False"
    ผลลัพธ์ที่พิมพ์ออกมาคืออะไร?
  2. [แบบทางเลือก] กำหนดให้ age = 15 และ height = 150
    ขั้นตอนวิธี: "ถ้าอายุมากกว่า 12 และ ความสูงตั้งแต่ 140 ขึ้นไป ให้พิมพ์คำว่า True ถ้าไม่ใช่ให้พิมพ์ False"
    ผลลัพธ์ที่พิมพ์ออกมาคืออะไร?
  3. [แบบทำซ้ำ] กำหนดให้ count = 1
    ขั้นตอนวิธี: "ในขณะที่ count น้อยกว่าหรือเท่ากับ 3 ให้เพิ่มค่า count ขึ้นทีละ 1 แล้ววนกลับไปตรวจสอบใหม่"
    เมื่อหลุดออกจากลูปทำซ้ำ ค่าของ count สุดท้ายจะมีค่าเท่ากับเลขใด?
  4. [แบบทำซ้ำ] กำหนดให้ sum = 0 และเริ่มต้นวนลูปโดยให้ตัวแปร i มีค่าตั้งแต่ 1 ถึง 3
    ขั้นตอนวิธี: "ในแต่ละรอบ ให้นำค่า i ไปบวกสะสมไว้ใน sum (รอบที่ 1 บวกลด้วย 1, รอบที่ 2 บวกด้วย 2, รอบที่ 3 บวกด้วย 3)"
    เมื่อสิ้นสุดการทำงาน ผลลัพธ์สุดท้ายในตัวแปร sum จะมีค่าเท่ากับเลขใด?
  5. [แบบทำซ้ำ + ทางเลือก] กำหนดให้ตรวจสอบตัวเลขตั้งแต่ 1 ถึง 5
    ขั้นตอนวิธี: "วนลูปตรวจสอบทีละเลข ถ้าเลขนั้นหารด้วย 2 ลงตัว (เป็นเลขคู่) ให้สะสมนับจำนวนไว้"
    เมื่อตรวจสอบครบถึงเลข 5 จำนวนเลขคู่ที่นับได้ทั้งหมดมีกี่จำนวน?
